# DSTURN AI Logo Update Instructions

## To update the logo with your new DSTURN AI design:

1. **Save your logo image** as `dsturn_ai_logo.png` in the `static/source/` directory
2. **Run the update script** below to replace all logo files

## Files that will be updated:
- `icon.png` - Main app icon (512x512 recommended)
- `icon_tray.png` - System tray icon (16x16 or 32x32)  
- `icon.ico` - Windows icon (multiple sizes)
- `icon.icns` - macOS icon (multiple sizes)
- `logo.png` - Additional logo file

## Manual steps if script doesn't work:
1. Copy your logo to the required sizes
2. Use an image converter to create .ico and .icns files
3. Replace the existing files in `static/source/`
